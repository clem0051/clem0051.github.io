Discription of Tools

All Pages: For all the pages, CSS and HTML was used to make the Navigation on the left, the header, and the background. These are constant on all the pages.

To-Do List: For this page, I used a table to hold the values of the items of the To-Do list. To create a new Item, a mix of HTML and Javascript was used to collect all the needed information. LocalStroage is used to save data locally for the website so that when you come back to the website, your data will be saved. Checkboxes are used for the complete catagory and layered function calls are used to make the remove button work properly.

Photo Gallery: For this page, I used a table to hold thumnails of all the images that are styled with CSS Sheets. Upon clicking on the images, a lightbox is created. This is done by using CSS Style Sheets, Javascript, and the use of Div's from HTML. The thumbnails are created by a table that spans across the bottom. Upon clicking on the thumbnails, the large image is changed to the appropriate image. The left right arrows along with the left and right arrow keys rotate the images. The play button plays a slideshow that is ran by javascript. Pressing the ESC button or clicking on the X in the upper righthand corner exits out of the lightbox.

Bio Page: Here, the use of CSS Style Sheets and HTML create the whole page. The video is embeded.
